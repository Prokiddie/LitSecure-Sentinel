## The Opportunity

Malawi has **over 12 million mobile subscribers**, a rapidly expanding digital financial ecosystem, and **zero nationally-deployed cybersecurity incident management infrastructure**. The Malawi Electronic Transactions and Cyber Security Act (2016) mandates incident reporting — yet no platform exists to receive, manage, or analyze those reports at scale.

**LitSecure Sentinel** fills this gap.

---

## What We Built

LitSecure Sentinel is a **full-stack national cybersecurity operations platform** — the first of its kind designed specifically for Malawi and the SADC region.

### Core Capabilities

| Module | Description |
|---|---|
| 🛡️ **National Incident Portal** | Citizens and organizations report cyber incidents via a secure public portal |
| 🧠 **AI Threat Analysis** | Gemini-powered incident classification, IOC extraction, and mitigation advice |
| 📡 **SOC Situation Room** | Real-time analyst workspace with live threat feeds and WebSocket collaboration |
| 📱 **SIM Swap Detection** | Clusters suspicious SIM replacement activity across mobile networks |
| 🌍 **Social Media Monitoring** | Scans for fraud, impersonation, and cyberattack chatter across platforms |
| 🗺️ **Malawi District Map** | Geographic incident heat mapping across all 28 districts |
| 🔔 **Real-Time Alerts** | SSE push notifications + Africa's Talking SMS for critical events |
| 🔐 **STIX/TAXII Intelligence** | Exports threat data in international sharing standards |
| 📚 **AI Knowledge Base** | Self-improving system that learns from analyst corrections |
| 👮 **9-Role RBAC** | Admin → Citizen, full access control matrix |

---

## The Problem Is Real

- **SIM Swap Fraud**: Malawi telcos lose an estimated **$5M–15M annually** to mobile money fraud
- **Phishing Campaigns**: Banking sector reports 3–8 major phishing waves per quarter
- **No Centralized Reporting**: Incidents go to different ministries, telcos, and police — siloed, delayed, unanalyzed
- **ETCSA Compliance Gap**: Organizations are legally required to report cyber incidents but have no standardized mechanism to do so
